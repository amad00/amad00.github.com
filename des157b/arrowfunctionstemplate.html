Factorial

This is a function that return the factorial of a number. I got this function from one of the examples I did in des157.

Factorial

            function factorial(num) {
                fac = num;
                for (let i = num - 1; i > 0; i--) {
                    fac *= i;
                }
                return fac;
            }

Function Expression

            const factorial = function (num) {
                fac = num;
                for (let i = num - 1; i > 0; i--) {
                    fac *= i;
                }
                return fac;
            }

Arrow Function

            const factorial = num => {
                fac = num;
                for (let i = num - 1; i > 0; i--) {
                    fac *= i;
                }
                return fac;
            }
